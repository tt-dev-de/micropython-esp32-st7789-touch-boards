## Adding More Fonts
When adding additional fonts to this directory, please include:

- **Source URL**  
- **Font family name**  
- **Copyright holder(s)**  
- **License name**  
- **SPDX identifier** (if available)  
- **Notes** (e.g., naming restrictions, special conditions)

This ensures consistent documentation and proper license compliance.
